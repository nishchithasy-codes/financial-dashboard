# 📊 Financial Dashboard

A Python project that fetches real-time financial data and 
visualizes it in an interactive dashboard.

## 🚀 Features
- 📈 Real-time stock price & volume charts
- 📰 Latest financial news feed
- 🏦 Economic indicators (GDP, CPI, Unemployment)

## 🛠️ APIs Used
- Yahoo Finance (yfinance) - Stock data
- News API - Financial news
- FRED API - Economic indicators

## 💻 Tech Stack
- Python
- Google Colab
- Matplotlib, Pandas

## ▶️ How to Run
1. Open the `.ipynb` file in Google Colab
2. Add your API keys in Colab Secrets
3. Run all cells
